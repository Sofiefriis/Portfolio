Versionsstyring



Hvad er versionsstyring

- og hvorfor er det relevant at bruge?


Bruges til at holde styr på forskellige versioner af samme dokument, og/eller flere delprojekter, og bygger på en metodisk tilgang. Dette kan være noget så simpelt som navngivning af filer og mapper. Versionsstyring bruges meget i forbindelse med kodning/programmering.

Det er relevant at bruge, da det gør det muligt at gå tilbage til en tidligere version af et dokument, hvis man fx fortryder sine ændringer. Man er sikker på altid at have backup. Det er også relevant i forbindelse med samarbejde med andre. Det er muligt at arbejde flere sammen om de samme dokumenter i software/webplatform som GitHub.



Hvad er Git?

Git er et versionsstyringssystem som sætter standarden for onlinetjenester som GitHub, GitLap, BitBucket osv.


GitHub


GitHub er et socialt medie for folk der gerne vil dele deres kode. Man kan oprette Git-repositories, en server i skyen, så det ikke er nødvendigt at installere én. Når filer er uploadet på GitHub, er de som udgangspunkt offentlige, og alle kan tilgå dem. Folk bruger det derfor også som en slags CV/portfolio. GitHub har en del features, til bl.a. projektstyring og samarbejde.
